# 📊 Data Professional Survey Analysis Dashboard

## Overview

This Power BI project analyzes real-world survey data collected from data professionals across different countries and job roles. The dashboard provides insights into salary trends, work-life balance, salary satisfaction, programming language preferences, and career experiences within the data industry.

## 🎯 Project Objectives

- Analyze survey responses from data professionals.
- Identify salary trends across different job roles.
- Understand work-life balance and salary satisfaction levels.
- Examine country-wise participation in the survey.
- Create an interactive dashboard for data-driven insights.

## 🛠️ Tools Used

- Power BI
- Power Query
- DAX
- Microsoft Excel

## 🧹 Data Preparation

The dataset was cleaned and transformed using Power Query by:

- Removing empty and unnecessary columns
- Applying filters and data cleaning techniques
- Splitting columns using delimiters
- Creating custom columns and formulas
- Renaming and transforming fields
- Preparing data for analysis and visualization

## 🔍 Key Insights

- Data Scientists reported the highest average salaries.
- Python emerged as the most preferred programming language.
- Most respondents reported moderate to high work-life balance satisfaction.
- Survey responses provided a global perspective on careers in data.
- Many professionals found entering the data field challenging but achievable.
