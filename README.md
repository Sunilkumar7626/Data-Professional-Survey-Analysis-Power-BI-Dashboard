📊 Data Professional Survey Analysis Dashboard
Overview

This Power BI project analyzes real-world survey data from data professionals across different countries and job roles. The dashboard provides insights into salary trends, work-life balance, programming language preferences, and career experiences within the data industry.

Project Objectives
Analyze survey responses from data professionals.
Identify salary trends across different job roles.
Understand work-life balance and salary satisfaction levels.
Examine country-wise participation in the survey.
Create an interactive dashboard for data-driven decision-making

Key Insights
Data Scientists reported the highest average salaries among surveyed professionals.
Python emerged as the most preferred programming language.
Most respondents reported moderate to high levels of work-life balance satisfaction.
Survey participants came from multiple countries, providing a global perspective on data careers.
Many respondents indicated that entering the data field was challenging but achievable.
